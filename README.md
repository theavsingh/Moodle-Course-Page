## 📖 Overview
This project is a responsive, student-friendly Moodle-like course dashboard built for Assignment 6. It presents course information, interactive announcements, a dynamic assignment tracker, and downloadable resources in a clean, organized layout.

## 🛠️ Technical Stack
* **HTML5:** Semantic document structure and dashboard layout.
* **CSS3:** Custom styling, interactive hover states, and mobile-responsive grid adjustments.
* **JavaScript:** Client-side interactivity, including toggleable announcements, dynamic stat counters, and assignment status updates.

## 📁 Files Included
* `index.html` - The main structure of the dashboard.
* `style.css` - All styling and responsive layout rules.
* `script.js` - The logic for interactive dropdowns, menu toggles, and dynamic dashboard statistics.
